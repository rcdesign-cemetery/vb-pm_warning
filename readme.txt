Description:
------------

vBulletin addon to show personal notifications prior to send PM. Useful for stuff members,
to advise read FAQ prior to ask questions.

Anyone can set their own warning. It will be shown, if someone wish to send private message.
User will see notification text and will be asked to confirm sending PM.

Released at vborg
http://www.vbulletin.org/forum/misc.php?do=producthelp&pid=pm_warning
